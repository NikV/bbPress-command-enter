Ever wanted keyboard shortcuts with bbPress?


== Description ==

This plugin gives you keyboard shortcuts to easily use with bbPress.



Keyboard Shortcuts Included:
- CMD + Enterb - To Submit a reply
- Control + Enter - To submit a reply
